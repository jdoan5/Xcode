INVOICE TRACKER — App Review Information

OVERVIEW
Invoice Tracker is a simple, fully offline personal record-keeping app for bills, invoices, and payments. The user adds records of money out (bills they pay) and money in (invoices/payments they receive), each with an amount, payee/client, category, payment method (online, check, card, cash, or bank transfer), an optional check number or payment/receipt URL, a reference number, issue and due dates, a status (Pending, Partially Paid, Paid; Overdue is derived from the due date), notes, and optional attachments. A dashboard summarizes totals and shows simple charts. All data is stored locally on the device using Apple's SwiftData (records and attachments) and UserDefaults (app preferences). There is no account, no server, no network use, no third-party SDK, no advertising, no analytics, and no tracking.

The app contains NONE of the special flows commonly asked about: no account registration, login, or deletion; no paid content, purchases, or subscriptions; and no user-generated content shared with or visible to other users (every record is private and stays on the device).

3. PURPOSE, AUDIENCE, PROBLEM, VALUE
Purpose: a calm, private way for individuals and very small businesses to keep track of the bills they pay and the invoices/payments they're owed. Problem it solves: people lose track of which bills are paid, which invoices are still outstanding, and when things are due, and often have no single private place that records the payment method (including check numbers) and a copy of the receipt. Invoice Tracker turns that into one organized, searchable ledger with clear statuses, due-date reminders, and a summary dashboard. Target audience: anyone who pays bills or sends invoices. The app is free, with no ads, no in-app purchases, no subscriptions, and no accounts.

4. SETUP, ACCESS, CREDENTIALS, SAMPLE FILES
No login, account, email, demo credentials, or sample files are required, and nothing needs to be configured. The app is fully usable the moment it launches. The Invoices tab opens ready to add a record (or the user can load sample data from Settings).

5. EXTERNAL SERVICES, TOOLS, OR PLATFORMS
None. The app makes no network requests and integrates no data providers, authentication services, payment processors, analytics, advertising, or crash-reporting SDKs. The only data is what the user enters, stored on the device.

6. PERMISSIONS
The app requests no permissions at launch. Three capabilities are used only when the user opts in:
• Notifications — if the user turns on "Due date reminders" in Settings, the app asks for notification permission at that moment and schedules LOCAL reminders for unpaid records with a due date. All notifications are created and delivered on the device; nothing is sent to a server. Declining simply disables reminders.
• Face ID / Touch ID — the user may turn on an optional app lock in Settings (Apple's LocalAuthentication). It only gates access to the app's own screen; the app does not access or store any biometric data.
• Photos / Files — to attach a receipt or invoice, the user picks a photo via the system Photos picker (PhotosPicker) or a file/PDF via the system document picker. These run out of process, so the app is never granted access to the full photo library or file system — only the items the user explicitly chooses are imported and stored locally.
The app uses no location, contacts, microphone, or camera.

7. REGIONAL DIFFERENCES
None. The app behaves identically in every region and storefront — free everywhere, with no region-locked content, features, or pricing. The user can choose their currency in Settings. The interface is in English.

8. FINANCIAL / REGULATED INDUSTRY
Invoice Tracker is a personal record-keeping and organization tool. It is NOT a financial institution, does NOT connect to any bank or account, does NOT move, send, or request money, and does NOT process payments. It only records information the user types in (a payment URL is stored as plain text the user can tap to open in their browser; the app itself performs no transactions). It provides no financial, investment, tax, or legal advice. Any figures shown in the App Store screenshots are fictitious examples created only for the screenshots.
