1. DEVICES AND OS TESTED
Simulator: iPhone 17, iPhone 17 Pro, iPhone 17 Pro Max, iPhone Air, iPhone 17e (all iOS 26.5).
Minimum deployment: iOS 17.0.

2. PURPOSE, AUDIENCE, PROBLEM, VALUE
LCD Classic Snake is a single-player retro snake game. The user steers a growing snake on a 20×20 grid, eating food pellets while avoiding walls and self-collision, on three difficulties (Easy / Medium / Hard) that change tick speed and per-food point value. A persistent named-player leaderboard tracks the top 10 scores per difficulty. Audience is casual / nostalgic players of any age. Free, no ads, no subscriptions, no in-app purchases.

3. SETUP, ACCESS, CREDENTIALS
No login, no account, no email, no demo credentials — the app has no server backend. On first launch a default player "Player 1" is seeded so a game can be started immediately. The reviewer can rename or delete that player and add others from the PLAYERS tab.

4. HOW TO REVIEW
Open the app → PLAY tab is shown by default → tap START GAME → the READY overlay appears → tap BEGIN → swipe up/down/left/right anywhere on the board to steer the snake → eat the orange pellet to score → hitting a wall or the snake's own body ends the game → tap PLAY AGAIN or QUIT. The SCORES tab shows the top 10 per difficulty. The PLAYERS tab manages player names. All three tabs work without a network connection.

5. EXTERNAL SERVICES / TOOLS / PLATFORMS
None. The app makes no network requests at any point and uses only first-party Apple frameworks:
- SwiftUI — user interface
- SwiftData — on-device persistence for players and scores

No third-party SDK of any kind. No analytics, no advertising, no auth provider, no payment processor, no cloud sync, no CloudKit, no Game Center.

6. REGIONAL DIFFERENCES
The app functions identically in every region and storefront. No region-locked features, no region-specific content, no region-specific pricing (free everywhere). UI is English only in this release; dates follow the device locale.

7. THIRD-PARTY MATERIAL
None. The classic "snake" gameplay is a generic mechanic with no trademarked elements. All visuals, code, and copy are original to this app. No third-party fonts, icons, or audio are included. No authorization or third-party credential is required to publish this app.
